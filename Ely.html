Aprendizajes del Año

En JavaScript, aprendí a declarar variables y a trabajar con diferentes tipos de datos, como números

y texto. También exploré cómo hacer condiciones utilizando if y else, así como repetir tareas con

bucles for. Aprendí a crear funciones para organizar mejor el código y a manipular el DOM para

modificar elementos en la página web. Además, trabajé con eventos, como los clics, para que la

página interactúe con el usuario.


En NetBeans, adquirí varias habilidades básicas y prácticas para programar en Java. Primero,

comprendí cómo utilizar la interfaz, incluyendo el menú principal y las ventanas para gestionar

proyectos y archivos. Luego, aprendí a crear proyectos y a organizar el código en carpetas, lo

que me ayudó a mantener un trabajo más ordenado.


También me enseñaron a escribir y ejecutar código Java directamente en el IDE, así como a utilizar

herramientas de depuración, como los puntos de interrupción, que me permiten detener el programa

en una línea específica para revisar lo que está sucediendo y corregir errores.


Otra parte interesante fue aprender a crear interfaces gráficas para aplicaciones de escritorio. En

NetBeans, esto es bastante sencillo, ya que puedo arrastrar y soltar botones, etiquetas y otros

elementos para diseñar una ventana. Además, trabajé con bases de datos para entender cómo

conectarlas a las aplicaciones y realizar consultas SQL desde NetBeans.


En la aplicación Dia, aprendí a hacer diagramas para organizar ideas y visualizar sus relaciones.

Pude crear diagramas de flujo, mapas conceptuales y organigramas utilizando diferentes formas y

flechas. También aprendí a guardar y exportar los diagramas para usarlos en otros documentos.

Aunque no es complicado, tuve que aprender a mover y conectar las figuras para que quedaran bien.


Aprendí lo básico de SQL, aunque no pude practicar en mi computadora porque no tenía el programa

instalado. Comprendí que SQL es un lenguaje utilizado para trabajar con bases de datos, donde

puedes crear tablas, agregar datos y realizar consultas para buscar información específica.
